# Cookiecutter NodeJS SNS Quick Start Application

A cookiecutter template to create a NodeJS SNS Quick Start Application using [Serverless Application Model (SAM)](https://github.com/awslabs/serverless-application-model).

## Requirements

* [AWS SAM CLI](https://github.com/awslabs/aws-sam-cli)

## Usage

Generate a boilerplate template in your current project directory using the following syntax:

* **NodeJS 12**: `sam init --runtime nodejs12.x --app-template quick-start-sns --name sns-app`

> **NOTE**: ``--name`` allows you to specify a different project folder name
