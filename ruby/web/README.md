# Cookiecutter Ruby Quick Start Web Application

A cookiecutter template to create a Ruby Quick Start Web Application using [Serverless Application Model (SAM)](https://github.com/awslabs/serverless-application-model).

## Requirements

* [AWS SAM CLI](https://github.com/awslabs/aws-sam-cli)

## Usage

Generate a boilerplate template in your current project directory using the following syntax:

* **Ruby {{ cookiecutter.options[cookiecutter.runtime].version }}**: `sam init --runtime {{ cookiecutter.runtime }}`

> **NOTE**: ``--name`` allows you to specify a different project folder name (`sam-app` is the default)
